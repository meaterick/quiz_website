파이썬 기초 문법

1. 변수와 데이터 타입

파이썬에서는 변수를 선언하고 데이터 타입을 지정할 필요가 없습니다. 자동으로 타입이 결정됩니다.

x = 10
name = "John"
is_valid = True

2. 주석

주석은 코드에 설명을 추가하는 데 사용됩니다. 한 줄 주석은 `#` 기호를 사용하며, 여러 줄 주석은 `'''` 또는 `"""`로 둘러싸인 텍스트 블록을 사용합니다.

# 이것은 한 줄 주석입니다. ''' 이것은 여러 줄 주석입니다. ''' """ 또는 이렇게도 사용할 수 있습니다. """

3. 조건문

if, elif, else 키워드를 사용하여 조건문을 작성합니다.

if 조건: # 조건이 참일 때 실행되는 코드 elif 다른조건: # 다른 조건이 참일 때 실행되는 코드 else: # 위의 모든 조건이 거짓일 때 실행되는 코드

4. 반복문

for와 while을 사용하여 반복문을 작성할 수 있습니다.

for 변수 in iterable: # 반복 실행될 코드 while 조건: # 조건이 참일 동안 실행될 코드

5. 함수

def 키워드로 함수를 정의하고, 매개변수와 반환값을 사용할 수 있습니다.

def 함수이름(매개변수): # 함수 내용 return 반환값

6. 리스트

순서가 있는 데이터를 담는 자료형입니다.

my_list = [1, 2, 3, 4, 5]
